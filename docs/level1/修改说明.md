## 关卡设计
敌人会分五次攻击（次数可定义），攻击使用json列表配置，每次攻击配置项包括攻击内容attack_words（多少字、每个字是什么，反击的字counterattack_words是什么）和节拍设计（每个字在开始的第几个节奏上出现beat_config），以及攻击结束后间隔多少个节拍interval_time会出现下一次攻击
``` JSON
{
    "attack_words":["你","好","笨"],
    "counterattack_words":["你","才","笨"],
    "damage_value":[1,1,1]
    "beat_config":[1,2,4],
    "interval_time":15
}
```
例如上述配置，攻击时发射的字是 你 好 笨，每个字会对应到counterattack_words的对应index的字 你 才 笨，反射到敌人身上分别造成的伤害值分别是1，1，1，每个字从出现的节拍点分别是这个攻击开始后的第1，2，4个节拍，此次攻击（第一个字出现）到下一次攻击的时间间隔是interval_time，也就是过15个节拍会出现下一次攻击。

游戏整体按节拍划分时间，节拍速度可以配置定义，每个节拍播放音效assets/music/tap.wav
